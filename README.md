# App description
## App functionality:
This is frontend application which takes a product list from Reqres Rest-Api ( https://reqres.in/ ) and displays it in the table on the page (5 products per page). There is a pagination and filtering (by product ID) functionality added. Pagination and filtering have the reflection in the URL of the page (so that makes it possible to send a link to the concrete data displayed on the page). After clicking on the product in the table, modal gets open (displaying the full data of the product).
## Technologies used in project:
React, Redux, React Router, Sass
## How to run the code:
* Download the code and instal dependencies (in the project directory run: `npm install`)
* Run the app in the development mode (In the project directory run: `npm start`, then open http://localhost:3000 in the browser)
